# iOS ↔ Desktop Two-Way Sync — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Two-way review sync between the iOS app and desktop Anki via Anki's own self-hosted sync server, so a review on either device shows up on the other with nothing lost or double-counted.

**Architecture:** Both apps are Anki sync *clients* pointed at one self-hosted `rslib` sync server (fronted by cloudflared for HTTPS to the phone). New work is small: 3 C-ABI sync wrappers in the Rust bridge, collection persistence + a Sync button on iOS, and desktop pointed at the server. No sync protocol is written and `rslib` is not modified — this reuses Anki's sync.

**Tech Stack:** Rust (`anki-bridge-rs` FFI crate, `prost`), Anki `rslib` sync (`SyncLogin`/`SyncCollection`/`FullUploadOrDownload`), Swift/SwiftUI (iOS), cloudflared, the built pyenv for the server.

`content/scripts/sync_smoketest.py` — collection A seeds the server (full upload), collection B pulls it (full download):

```python
import os, tempfile
from anki.collection import Collection
from anki.sync import SyncAuth
import anki.sync_pb2 as pb

ENDPOINT = "http://127.0.0.1:8080/"   # trailing slash required
USER, PASS = "demo", "demo"

def open_col():
    d = tempfile.mkdtemp()
    return Collection(os.path.join(d, "c.anki2"))

def login(col):
    return col._backend.sync_login(username=USER, password=PASS, endpoint=ENDPOINT)

def sync(col, auth):
    out = col._backend.sync_collection(auth=auth, sync_media=False)
    req = out.required  # ChangesRequired: 0 none,1 normal,2 full,3 full_download,4 full_upload
    if req in (pb.SyncCollectionResponse.FULL_UPLOAD, pb.SyncCollectionResponse.FULL_SYNC):
        col._backend.full_upload_or_download(auth=auth, upload=True, server_usn=out.server_media_usn)
    elif req == pb.SyncCollectionResponse.FULL_DOWNLOAD:
        col._backend.full_upload_or_download(auth=auth, upload=False, server_usn=out.server_media_usn)
    return req

# A: add a note, seed the server
a = open_col()
m = a.models.by_name("Basic"); note = a.new_note(m); note.fields[0] = "sync-probe"; a.add_note(note, a.decks.id("Default"))
print("A notes:", a.note_count())
print("A sync required:", sync(a, login(a)))
a.close()

# B: fresh collection, pull from server
b = open_col()
print("B notes before:", b.note_count())
print("B sync required:", sync(b, login(b)))
print("B notes after:", b.note_count())
assert b.note_count() >= 1, "B did not receive A's note — sync round-trip FAILED"
print("ROUND-TRIP OK")
b.close()
```

- [ ] **Step 2: Add `anki_sync_login`** (after lib.rs:599)

```rust
/// Log in to a sync server. Returns a serialized SyncAuth (caller frees via anki_free_response).
/// # Safety: standard FFI pointer contract; strings are NUL-terminated UTF-8.
#[no_mangle]
pub unsafe extern "C" fn anki_sync_login(
    backend_ptr: i64,
    endpoint: *const c_char,
    user: *const c_char,
    pass: *const c_char,
    out_data: *mut *mut u8,
    out_len: *mut usize,
) -> c_int {
    if backend_ptr == 0 || user.is_null() || pass.is_null() { return -1; }
    let Some(username) = (unsafe { cstr_to_string(user) }) else { return -1; };
    let Some(password) = (unsafe { cstr_to_string(pass) }) else { return -1; };
    let endpoint = unsafe { cstr_to_string(endpoint) }.unwrap_or_default();
    let req = anki_proto::sync::SyncLoginRequest {
        username,
        password,
        endpoint: if endpoint.is_empty() { None } else { Some(endpoint) },
    };
    let input = req.encode_to_vec();
    let backend = unsafe { &*(backend_ptr as *const Backend) };
    match backend.run_service_method(SVC_BACKEND_SYNC, M_SYNC_LOGIN, &input) {
        Ok(output) => { unsafe { set_output(output, out_data, out_len) }; 0 }
        Err(err_bytes) => { unsafe { set_output(err_bytes, out_data, out_len) }; 1 }
    }
}
```

- [ ] **Step 3: Add `anki_sync_collection`** (directly below)

```rust
/// Run a collection sync using a serialized SyncAuth from anki_sync_login.
/// Returns a serialized SyncCollectionResponse (inspect `.required` for full-sync).
/// # Safety: auth_data/auth_len describe a valid SyncAuth protobuf.
#[no_mangle]
pub unsafe extern "C" fn anki_sync_collection(
    backend_ptr: i64,
    auth_data: *const u8,
    auth_len: usize,
    out_data: *mut *mut u8,
    out_len: *mut usize,
) -> c_int {
    if backend_ptr == 0 || auth_data.is_null() || auth_len == 0 { return -1; }
    let auth_bytes = unsafe { slice::from_raw_parts(auth_data, auth_len) };
    let auth = match anki_proto::sync::SyncAuth::decode(auth_bytes) { Ok(a) => a, Err(_) => return -1 };
    let req = anki_proto::sync::SyncCollectionRequest { auth: Some(auth), sync_media: false };
    let input = req.encode_to_vec();
    let backend = unsafe { &*(backend_ptr as *const Backend) };
    match backend.run_service_method(SVC_BACKEND_SYNC, M_SYNC_COLLECTION, &input) {
        Ok(output) => { unsafe { set_output(output, out_data, out_len) }; 0 }
        Err(err_bytes) => { unsafe { set_output(err_bytes, out_data, out_len) }; 1 }
    }
}
```

```rust
/// Full upload or download (used when SyncCollectionResponse.required is a full-sync variant).
/// upload=true -> full_upload; false -> full_download. server_usn < 0 omits media sync.
/// # Safety: auth_data/auth_len describe a valid SyncAuth protobuf.
#[no_mangle]
pub unsafe extern "C" fn anki_full_upload_or_download(
    backend_ptr: i64,
    auth_data: *const u8,
    auth_len: usize,
    upload: bool,
    server_usn: i32,
    out_data: *mut *mut u8,
    out_len: *mut usize,
) -> c_int {
    if backend_ptr == 0 || auth_data.is_null() || auth_len == 0 { return -1; }
    let auth_bytes = unsafe { slice::from_raw_parts(auth_data, auth_len) };
    let auth = match anki_proto::sync::SyncAuth::decode(auth_bytes) { Ok(a) => a, Err(_) => return -1 };
    let req = anki_proto::sync::FullUploadOrDownloadRequest {
        auth: Some(auth),
        upload,
        server_usn: if server_usn < 0 { None } else { Some(server_usn) },
    };
    let input = req.encode_to_vec();
    let backend = unsafe { &*(backend_ptr as *const Backend) };
    match backend.run_service_method(SVC_BACKEND_SYNC, M_FULL_UPLOAD_OR_DOWNLOAD, &input) {
        Ok(output) => { unsafe { set_output(output, out_data, out_len) }; 0 }
        Err(err_bytes) => { unsafe { set_output(err_bytes, out_data, out_len) }; 1 }
    }
}
```
